一.寻找喜欢的主题

可以在https://github.com/hexojs/hexo/wiki/Themes 这里找到自己喜欢的主题

我自己找到了一款更好看的

以我自己的安装过程为例:

二.安装主题:

到hexo所在的路径下E:hexo,右键打开Git Bush,执行:

$ git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

三.使用主题:

打开_config.yml做如下修改:

theme: yilia
